UPDATES
1. Increment version in pyproject.toml

2. Remove old distributions
rm -rf dist/*

3. Build new distribution
python3 -m build

4. Upload the new version to PyPI using twine
python3 -m twine upload dist/*

5. Update local package
pip install --upgrade dreams_core
